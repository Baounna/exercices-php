# 🐘 PHP Exercises

A collection of PHP practice exercises focused on form handling and basic computation, built while learning server-side PHP.

## 📋 Contents

| File | Description |
|------|-------------|
| `exercices1.php` | Student grade form — collects name, age & 3 grades, then computes sum, average and result |
| `exercice2.php` | Variation of the grade-management form with additional logic |
| `exercice3.php` | PHP exercise — form processing |
| `exercice4.php` | PHP exercise — form processing |

## 🛠️ Built With

- **PHP** — server-side processing
- **HTML** — forms and structure

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/Baounna/exercices-php.git
cd exercices-php

# Start PHP's built-in server
php -S localhost:8000

# Open http://localhost:8000/exercices1.php in your browser
```

> Requires PHP installed locally.
